DisclosureDiff

AI chat invents company “facts.” Reading two 10-Ks by hand is painful. DisclosureDiff plugs into Cursor or Claude, compares real SEC filings, and always links back to EDGAR — so you can see what actually changed.

View on GitHub Docs site Install

Why you’d use this

Use it when you care what a company wrote in its filings — new risks in a 10-K (the annual SEC report), whether a claim matches the text, or how peers’ risk themes differ — not stock prices or trading tips.

You might ask: what new risks did Apple add this year? Does Microsoft’s latest 10-K support a homework claim? How do NVIDIA and AMD risk themes compare? Or wire it into an agent that must not invent filing text.

You get structured changes, short excerpts, and live sec.gov links. Not a stock picker, not a price feed, not financial advice.

Try these prompts

  1. Diff Apple’s Item 1A risk factors year-over-year and cite every change with EDGAR links.
  2. Ground this claim against Microsoft’s latest 10-K: “Microsoft discloses material cybersecurity risks.” Include excerpts and filing URLs.
  3. Compare NVDA vs AMD Item 1A themes — which risks show up for one peer but not the other? Cite EDGAR.

Install (Cursor / Claude)

{
  "mcpServers": {
    "disclosurediff": {
      "command": "node",
      "args": ["/path/to/disclosurediff-mcp/dist/index.js"],
      "env": {
        "SEC_CONTACT_EMAIL": "you@example.com"
      }
    }
  }
}

Clone: https://github.com/subasah/disclosurediff-mcp.git. Landing: https://subasah.github.io/disclosurediff-mcp/. Remote MCP: deploy per the repo DEPLOY.md, then point url at your HTTPS /mcp with header X-API-Key (no public host published yet).

Architecture

Agent → Streamable HTTP / stdio MCP → EDGAR client (User-Agent + throttle + cache) → section diff / claim ground → citation gate → audit log.

Not financial advice. Summarizes public SEC text only. Verify every claim on EDGAR. See methodology tool and threat model in the repo.
