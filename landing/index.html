DisclosureDiff

Year-over-year disclosure changes with mandatory filing citations — built for agents that must not invent financial facts.

View on GitHub Docs site Install

Try these prompts

  1. Diff Apple’s Item 1A risk factors year-over-year and cite every change with EDGAR links.
  2. Ground this claim against Microsoft’s latest 10-K: “Microsoft discloses material cybersecurity risks.”
  3. Compare NVDA vs AMD vs AVGO Item 1A themes — which peers discuss climate risk?

Install (Cursor / Claude)

{
  "mcpServers": {
    "disclosurediff": {
      "command": "node",
      "args": ["/path/to/disclosurediff-mcp/dist/index.js"],
      "env": {
        "SEC_CONTACT_EMAIL": "you@example.com"
      }
    }
  }
}

Clone: https://github.com/subasah/disclosurediff-mcp.git. Landing: https://subasah.github.io/disclosurediff-mcp/. Remote MCP: deploy per the repo DEPLOY.md, then point url at your HTTPS /mcp with header X-API-Key (no public host published yet).

Architecture

Agent → Streamable HTTP / stdio MCP → EDGAR client (User-Agent + throttle + cache) → section diff / claim ground → citation gate → audit log.

Not financial advice. Summarizes public SEC text only. Verify every claim on EDGAR. See methodology tool and threat model in the repo.
